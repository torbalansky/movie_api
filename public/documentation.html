Documentation page

This application is intended to guide you through the realm of the finest movies ever produced!

  • You can browse details about various movies, directors, and genres here.
  • You can also register, modify your personal information, and compile a list of your preferred movies.

Table of the endpoints and their corresponding request methods:

Request URL HTTP Method Response
Returns a list of all movies /movies GET A JSON object containing data about all movies
Returns data about a single movie by title /movies/title GET A JSON object containing data about the movie's description, genre, director, image URL, and whether it's featured or not
Returns data about a genre by name /movies/title/genreName GET A JSON object containing data about the genre's description
Returns data about a director by name /movies/title/directorName GET A JSON object containing data about the director's bio, birth year, and death year
Allows new users to register /users POST A text message indicating successful registration of the account
Allows users to add a movie to their list of favorites /users/id/favoriteMovieTitle POST A text message indicating that the movie has been added to the user's list of favorites
Allows users to update their user info /users/id PUT A text message indicating successful update of user info
Allows users to remove a movie from their list of favorites /users/id/favoriteMovieTitle DELETE A text message indicating that the movie has been removed from the user's list of favorites
Allows existing users to deregister /users/id DELETE A text message indicating successful deletion of the user account
